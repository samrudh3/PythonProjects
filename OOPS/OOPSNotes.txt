Getter and setter :

Access Modifiers in python : Access specifiers/Modifiers are used to limit the access of property/method outside the class
Public,private, protected 
Public : All the variables and methods in python are pubic by default, any variable followed by self keyword is public
private : in python there is not such things, but we can indicate the variable by "__"  Ex: __name
    Called Name Mangling is a technique to protect the private variable to be ovewritten by subclass
protected : can be access by class and its subclass, name covention is followed by "_" Ex: _name
